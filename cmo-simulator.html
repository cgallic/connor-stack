Run this CMO Harness session locally

You don't need a custom UI to run this loop. Paste this configuration block directly into your own Claude Code terminal workspace (with `kai-cmo-harness` installed) to execute the Copywriter, QA, and Delivery agents locally on your code files. 

I need to run the CMO Harness pipeline.
1. Load project memory from ~/.claude/projects/connor-stack/memory/
2. Ingest the campaign specs from the active file
3. Run the copywriter skill using the Ralph Wiggum persona settings
4. Run the QA Auditor tool and calculate the Four U's scorecard
5. Format the final output as a responsive HTML artifact file and save it to templates/html-artifacts/
